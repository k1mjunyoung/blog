```
ISP
└──모뎀: 공인IP 할당
    ├── PC(공인IP: 220.221.222.1)
    ├── PC(공인IP: 220.221.222.2)
    └── 공유기(공인IP: 220.221.222.3, 사설IP: 192.168.0.1)
        ├── PC(사설IP: 192.168.0.2)
        └── TV(사설IP: 192.168.0.3)
```

## 모뎀

-   공인IP 할당

## 공유기

-   사설 네트워크 형성
-   사설IP 부여
-   라우터(L3, 개념적으로 게이트웨이) 역할
-   DHCP
-   LAT

## 외부에서 내 컴퓨터로 접속?

외부 -> ISP -> 모뎀 -> 공유기(220.221.223.3:80 - 192.168.0.2:80 포트포워딩) -> PC(192.168.0.2:80)

## 허브와 스위치

### 허브

-   OSI L1
-   IP주소 할당하는 기능 X
-   연결된 모든 기기로 전송

### 스위치

-   OSI L2
-   IP주소 할당하는 기능 X
-   기기들의 MAC주소 저장
-   브로드캐스트(모든 기기들에게 목적지가 당신인지 확인)
